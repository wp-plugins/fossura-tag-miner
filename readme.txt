Enrich your blog posts with automatically added, relevant tags.

== Description ==

Tag Miner uses the textCavate API to extract and add the most relevant tags to your blog posts. Simply write, publish and voila!

Choose to consider all possible tags equally, or to give precedence to the names of people, places and things.

The following languages are currently supported:

*	Afrikaans 
*	English
* 	German

== Installation ==
= The painless way =

*	Go to "Plugins" > "Add"
*	Search for "tag miner"
*	Click "Install now"
*	Activate the plugin

A new menu ("Tag Miner") will appear in your Settings section

= The manual way =

*	Download the plugin .zip file by clicking the red download button at the top right of this page
*	Go to "Plugins" > "Add"
*	Select "Upload"
*	Browse to and upload the plugin .zip file
*	Activate the plugin

A new menu ("Tag Miner") will appear in your Settings section.

== Frequently Asked Questions ==
= How do I activate the plugin? =
Register for a free account at http://www.textcavate.com and follow the activation steps in your plugin settings panel.

= How do I get the tags? =
Simply publish or save your post. The plugin will take care of the rest.

= What if I don't like the tags? =
We really hope you like them, but if you don't, you can simply delete the auto-generated tags or manually add different ones.

= Can you add feature X? =
We'd love to hear your idea! Please rate and review the plugin, and mention your suggestion in the review text.
